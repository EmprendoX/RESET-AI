# PRD — AI Coach App Independiente

Producto standalone MVP v1.0. Ver documento completo en el plan de implementación del proyecto.

## Resumen

AI Coach es una app independiente con:
- Auth propia (Supabase)
- Onboarding 6 pasos
- Chat con IA + tools
- Memoria CRUD con consentimiento
- Metas, recomendaciones, check-ins
- Eventos internos + audit logs
- Panel admin + feature flags
- Arquitectura clonable

## Principios

1. **Independencia total** — no depende de otras plataformas
2. **Clonabilidad** — core separado de branding
3. **Memoria controlada** — el usuario decide
4. **No inventar datos** — solo contexto real
5. **Acción antes que teoría**

## Definition of Done

El MVP está listo cuando los 21 criterios del PRD §34 se cumplen.
